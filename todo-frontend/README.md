# Todo Application

This is a simple Todo application built with React and TypeScript. The application allows users to manage their todo items, including adding, toggling, and displaying them.

## Project Structure

```
todo-frontend
├── src
│   ├── components
│   │   └── TodoItem.tsx
│   ├── pages
│   │   └── Home.tsx
│   ├── App.tsx
│   ├── index.tsx
│   └── types
│       └── index.ts
├── public
│   └── index.html
├── package.json
├── tsconfig.json
└── README.md
```

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd todo-frontend
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   npm start
   ```

The application will be available at `http://localhost:3000`.

## Features

- Add new todo items
- Toggle completion status of todo items
- Display a list of todos

## Technologies Used

- React
- TypeScript
- CSS
